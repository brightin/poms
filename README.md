# Poms

The Poms gem provides an interface to the Dutch public broadcaster API: POMS. It
is a couchdb service that is publicly available.

## Installation

Add this line to your application's Gemfile:

    gem 'poms'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install poms
